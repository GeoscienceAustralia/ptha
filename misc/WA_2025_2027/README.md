# WA tsunami modelling (Kalbarri to Onslow region) 2025-2027

Key subfolders:

* [kalbarri_2_onslow](kalbarri_2_onslow) - A preliminary experimental version of the modelling, covering the full area but at lower resolution and without breakwalls/inverts/etc. This was used to get the code working with variable rigidity PTHA18 scenarios, de-risk the main calculations implemented in other folders, check the sensitivity of the results to including horizontal components in the tsunami sources, and test the tidal adjustment technique.

* [kalbarri_2_coralbay](kalbarri_2_coralbay) - Model covering the region from (somewhat south of) Kalbarri to Coral Bay, with high resolution in towns.
