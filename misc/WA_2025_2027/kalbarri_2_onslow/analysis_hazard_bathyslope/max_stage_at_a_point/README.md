Comparison of PTHA18 max-stages with the Monte Carlo sample.

See `../../analysis_hazard/max_stage_at_a_point/README.md` for more details. Compared to calculations in the latter folder, the Nonlinear Simulations here use different earthquake sources (including the effect of horizontal components on sloping bathymetry in the PTHA18 sources). Thus the nonlinear simulation results SHOULD NOT AGREE WITH PTHA18 unless the effect of horizontal components is small.
